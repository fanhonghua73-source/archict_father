## 问题排查

如果页面还是空白：
1. 按 F12 打开浏览器控制台，查看是否有错误
2. 检查 Network 标签，确认 API 请求是否成功
3. 清除浏览器缓存（Ctrl+Shift+Delete）
4. 重启后端服务：`cd backend && python app.py`
5. 刷新浏览器页面

如果新增项目没反应：
1. 检查浏览器控制台是否有 JavaScript 错误
2. 确认后端服务正在运行（http://localhost:5002）
3. 检查 Network 标签，查看 POST /api/projects 请求是否成功
