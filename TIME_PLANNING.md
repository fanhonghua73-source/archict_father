# 管理台人员时间规划功能说明

## 已添加的功能

### 1. 人员时间规划展示区域

在管理台右侧添加了"人员时间规划"区域，位于经济技术指标上方。

**布局结构**：
- 左列：文件夹层级（200rpx）
- 中列：文件表格 + 人员格子（1fr）
- 右列：人员时间规划 + 经济技术指标（280rpx）

**显示逻辑**：
- 只显示**激活的人员**（active = true）
- 显示人员的姓名、角色、时间安排
- 如果没有时间安排，显示"暂无时间安排"
- 如果没有激活人员，显示"暂无激活人员"

### 2. 人员时间规划数据结构

```javascript
{
  "id": 1,
  "name": "张三",
  "busy": 75,
  "active": true,
  "role": "主创建筑师",
  "schedule": "2026-03-01 至 2026-03-15: 方案设计\n2026-03-16 至 2026-03-31: 施工图绘制"
}
```

### 3. 编辑人员时间安排

**方式1：点击人员格子的编辑按钮（✎）**
1. 在中列的人员格子上，点击右上角的编辑按钮
2. 弹出编辑弹窗
3. 可以编辑：
   - 姓名
   - 忙碌度（滑块 0-100%）
   - 角色（如：主创建筑师、项目经理）
   - 时间安排（多行文本）
4. 点击"保存"按钮

**时间安排格式示例**：
```
2026-03-01 至 2026-03-15: 方案设计
2026-03-16 至 2026-03-31: 施工图绘制
2026-04-01 至 2026-04-15: 初步设计
```

### 4. 激活/停用人员

**点击人员格子**：
- 单击人员格子（不是编辑按钮）可以切换激活状态
- 激活的人员：绿色边框，浅绿背景
- 停用的人员：灰色边框，白色背景
- 只有激活的人员会显示在右侧的"人员时间规划"区域

### 5. 文件类型选择改进

**上传文件时的文件类型选择**：
- 方式1：点击下拉选择器，从预设类型中选择（DWG、PDF、SKP、JPG、PNG、DOC、XLS）
- 方式2：如果下拉选择器不工作，可以在下方的输入框中手动输入文件类型

**注意**：UniApp 的 picker 组件在浏览器预览中可能不工作，建议使用真机或模拟器测试。

## 使用流程

### 场景1：为新项目分配人员

1. 进入管理台页面
2. 在中列的人员格子中，点击需要参与项目的人员
3. 人员格子变为绿色（激活状态）
4. 右侧"人员时间规划"区域会显示该人员
5. 点击人员格子的编辑按钮（✎）
6. 填写角色和时间安排
7. 保存

### 场景2：查看项目人员安排

1. 进入管理台页面
2. 查看右侧"人员时间规划"区域
3. 可以看到所有激活人员的：
   - 姓名
   - 角色
   - 时间安排

### 场景3：调整人员时间安排

1. 点击人员格子的编辑按钮（✎）
2. 修改时间安排
3. 保存
4. 右侧"人员时间规划"区域自动更新

### 场景4：移除项目人员

1. 点击人员格子（不是编辑按钮）
2. 人员格子变为灰色（停用状态）
3. 该人员从右侧"人员时间规划"区域消失

## 样式说明

### 人员时间规划卡片

- 背景：白色
- 边框：1px #e0e0e0
- 圆角：8rpx
- 内边距：16rpx
- 间距：12rpx

### 人员信息

- 姓名：22rpx，加粗，黑色
- 角色：20rpx，绿色文字，浅绿背景，圆角标签
- 时间安排：20rpx，灰色，支持多行显示

### 空状态

- 无激活人员时：显示"暂无激活人员"
- 无时间安排时：显示"暂无时间安排"（斜体，灰色）

## 数据持久化

所有人员信息（包括时间安排）都会自动保存到后端：
- 点击人员格子切换激活状态 → 立即保存
- 编辑人员信息并点击保存 → 保存到后端
- 数据存储在：`backend/data/management/project_{id}.json`

## 技术实现

### Computed 属性

```javascript
activeStaffList() {
    // 只显示激活的人员
    return this.staffList.filter(s => s.active)
}
```

### 模板

```vue
<view class="time-planning-section">
    <view class="section-title">人员时间规划</view>
    <scroll-view class="planning-list" scroll-y>
        <view v-for="(staff, index) in activeStaffList" :key="index" class="planning-item">
            <view class="planning-header">
                <text class="planning-name">{{ staff.name }}</text>
                <text class="planning-role">{{ staff.role || '未设置角色' }}</text>
            </view>
            <view v-if="staff.schedule" class="planning-schedule">
                <text>{{ staff.schedule }}</text>
            </view>
            <view v-else class="planning-empty">
                <text>暂无时间安排</text>
            </view>
        </view>
    </scroll-view>
</view>
```

## 注意事项

1. 时间安排支持多行文本，使用 `\n` 换行
2. 只有激活的人员才会显示在时间规划区域
3. 人员格子的编辑按钮使用 `@click.stop` 防止触发格子的点击事件
4. 时间规划区域使用 `scroll-view` 支持滚动
5. 右列使用 flex 布局，时间规划和经济指标各占一定比例
