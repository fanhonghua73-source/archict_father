## 在 HBuilderX 中运行

1. 打开 HBuilderX
2. 文件 → 打开目录
3. 选择: uniapp-frontend
4. 运行 → 运行到浏览器 → Chrome

## 页面导航

- 底部 TabBar 可切换 5 个主要页面
- 左侧导航可点击切换页面
- 工作台的"图片生成"可跳转到图片生成页面

## 后端连接

- API 地址: http://localhost:5002
- 项目页面会自动加载后端数据
- 如果后端未启动，使用模拟数据
